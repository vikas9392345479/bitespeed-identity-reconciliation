# Bitespeed Identity Reconciliation Service

A backend web service built with **Node.js, Express, and SQLite (Sequelize)** to reconcile multiple customer contact points into a single "Primary" identity.

## 🚀 Features
- **Identity Linking**: Automatically links new purchases to existing customers using email or phone number.
- **Primary-Secondary Logic**: Maintains a single source of truth with one "Primary" contact and multiple "Secondary" links.
- **Relational Database**: Uses a SQL-based approach (SQLite) for structured data integrity.

## 🛠️ Tech Stack
- **Backend**: Node.js, Express
- **Database**: SQLite (via Sequelize ORM)
- **Language**: JavaScript
